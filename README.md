# Red-Team Agent Spec

This repository contains a read-only red-team agent operating specification.

The spec defines the agent's role, boundaries, evidence standards, and stop
conditions for reviewing upstream-readiness workflows. It is intended to help
keep red-team review independent from implementation, approval, and publishing
roles.

## Contents

- `red-team-agent-spec.md` — the operating specification

## Notes

This repository does not contain implementation code, credentials, private
configuration, audit findings, or project-specific evidence logs.

The spec is not an approval record and does not represent any upstream project
policy.
